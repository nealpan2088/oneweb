## 🚀 快速部署指南

### **部署新临时小程序**
```bash
# 1. 创建临时目录
cd /home/admin/yunxinwangdian/
mkdir -p temp/my-demo/

# 2. 上传代码
# 将文件复制到 temp/my-demo/

# 3. 添加Nginx配置
sudo sed -i '/^}$/i\    # 临时小程序: my-demo\n    location /temp/my-demo/ {\n        alias /home/admin/yunxinwangdian/temp/my-demo/;\n        index index.html;\n    }' /etc/nginx/conf.d/yunxinwangdian.conf

# 4. 应用配置
sudo nginx -t && sudo systemctl reload nginx

# 5. 验证
curl -I http://localhost/temp/my-demo/
```

## 🗑️ 清理管理

### **临时小程序生命周期**
```
创建 → 使用 → 归档/删除
```

### **清理脚本**
```bash
#!/bin/bash
# cleanup-temp-app.sh
APP_NAME=$1

if [ -z "$APP_NAME" ]; then
    echo "用法: $0 <小程序名称>"
    echo "示例: $0 v2  (清理/temp/v2/)"
    exit 1
fi

echo "🗑️  清理临时小程序: $APP_NAME"

# 1. 备份（可选）
BACKUP_DIR="/home/admin/yunxinwangdian/temp-backup/$APP_NAME-$(date +%Y%m%d)"
mkdir -p "$BACKUP_DIR"
cp -r "/home/admin/yunxinwangdian/temp/$APP_NAME/"* "$BACKUP_DIR/" 2>/dev/null || true
echo "📦 已备份到: $BACKUP_DIR"

# 2. 删除文件
rm -rf "/home/admin/yunxinwangdian/temp/$APP_NAME"
echo "📁 文件已删除"

# 3. 移除Nginx配置
sudo sed -i "/location \\/temp\\/$APP_NAME\\/ {/,/}/d" /etc/nginx/conf.d/yunxinwangdian.conf
sudo nginx -t && sudo systemctl reload nginx
echo "🔧 Nginx配置已移除"

echo "✅ 清理完成"
echo "⚠️  备份保留7天，如需永久删除: rm -rf $BACKUP_DIR"
```

### **定期清理策略**
```bash

# - 超过90天: 删除
```

## 🔒 安全考虑

### **临时小程序特点**
1. **独立性强** - 不影响主站
2. **生命周期短** - 使用后及时清理
3. **权限宽松** - 便于快速部署
4. **无数据持久化** - 不存储重要数据

### **安全建议**
```bash
# 1. 限制访问（如果需要）
location /temp/private/ {
    alias /home/admin/yunxinwangdian/temp/private/;
    index index.html;
    
    # 添加基础认证
    auth_basic "Restricted";
    auth_basic_user_file /etc/nginx/.htpasswd;
}

# 2. 创建.htpasswd文件
echo "username:$(openssl passwd -crypt password)" | sudo tee /etc/nginx/.htpasswd
```

## 🔧 工具脚本

### **管理面板脚本**
```bash
#!/bin/bash
# temp-apps-manager.sh
echo "🎪 临时小程序管理面板"
echo "========================"

while true; do
    echo ""
    echo "1. 列出所有临时小程序"
    echo "2. 创建新小程序"
    echo "3. 删除小程序"
    echo "4. 查看访问统计"
    echo "5. 退出"
    echo ""
    read -p "请选择: " choice
    
    case $choice in
        1)
            echo "📁 临时小程序列表:"
            ls -la /home/admin/yunxinwangdian/temp/ 2>/dev/null || echo "  暂无小程序"
            ;;
        2)
            read -p "请输入小程序名称: " app_name
            mkdir -p "/home/admin/yunxinwangdian/temp/$app_name"
            echo "✅ 已创建: /home/admin/yunxinwangdian/temp/$app_name/"
            echo "🌐 访问地址: http://47.110.156.11/temp/$app_name/"
            ;;
        3)
            read -p "请输入要删除的小程序名称: " app_name
            ./cleanup-temp-app.sh "$app_name"
            ;;
        4)
            echo "📊 访问统计:"
            for app in /home/admin/yunxinwangdian/temp/*/; do
                app_name=$(basename "$app")
                count=$(grep -c "/temp/$app_name/" /var/log/nginx/access.log 2>/dev/null || echo "0")
                echo "  $app_name: $count 次访问"
            done
            ;;
        5)
            echo "再见！"
            exit 0
            ;;
        *)
            echo "无效选择"
            ;;
    esac
done
```

### **自动清理脚本**
```bash
#!/bin/bash
# auto-cleanup.sh
echo "🧹 自动清理临时小程序"
echo "========================"

# 清理超过30天未修改的小程序
find /home/admin/yunxinwangdian/temp/ -maxdepth 1 -type d -mtime +30 | while read dir; do
    app_name=$(basename "$dir")
    if [ -n "$app_name" ] && [ "$app_name" != "temp" ]; then
        echo "清理: $app_name (超过30天未修改)"
        ./cleanup-temp-app.sh "$app_name"
    fi
done

echo "✅ 自动清理完成"
```
