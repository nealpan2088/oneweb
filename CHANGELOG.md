# 更新日志 - 云信旺店

> 所有重要的项目变更都将记录在此文件中

## [1.1.0] - 2026-03-22

### ✨ 新增
- **门店类型** - 支持咖啡馆、餐厅、零售、美容、健身等多种店铺类型
- **装修主题** - 支持温馨、极简、轻奢、清新、复古、可爱、科技等多种主题
- **品牌配置** - 支持自定义品牌 Logo、配色方案
- **功能推荐** - 根据门店类型自动推荐功能模块
- **版本管理** - 添加数据库版本表、VERSION 文件

### 🔧 优化
- 数据库结构优化，添加新字段索引
- 现有店铺自动设置为"咖啡馆"类型 + "温馨"主题

### 📝 说明
- 代号：万象更新
- 完全向后兼容，不影响现有功能
- 数据库已升级至 v1.1.0

## 版本规范

本项目采用 [语义化版本](https://semver.org/lang/zh-CN/)：

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

**格式：** `主版本号。次版本号。修订号`

**示例：** `1.1.0`
