问题描述

错误: GET http://47.110.156.11/api/v1/shops/3 502 (Bad Gateway)

原因: 前端请求路径与后端API路径不匹配

  • 前端请求: /api/v1/shops/3
  • 后端实际: /api/shop/3
  • Nginx配置: 代理 /api/shop/ 到后端

🔄 修复内容

// 修复前 (api.js) getBaseURL() { return `${window.location.origin}/api/v1`; } // 修复后 (api.js) getBaseURL() { return `${window.location.origin}/api/shop`; } // 修复前 (shopService.js) async getShopInfo(shopId) { return await this.apiClient.get(`/shops/${shopId}`); } // 修复后 (shopService.js) async getShopInfo(shopId) { return await this.apiClient.get(`/${shopId}`); }
已修复 api.js 已修复 shopService.js Nginx配置正确
