# IT技术服务站

一个专业的IT技术服务展示网站，提供电脑维修、网页开发和IT技术支持服务。

## 🚀 项目概述

### 项目目标
- 展示专业的IT技术服务
- 提供在线咨询和预约功能
- 建立可信赖的技术服务品牌形象
- 吸引潜在客户并促进业务转化

### 核心功能
1. **服务展示** - 清晰展示电脑维修、网页开发、IT技术服务
2. **在线咨询** - 客户可以直接提交咨询需求
3. **成功案例** - 展示过往的成功项目
4. **响应式设计** - 适配所有设备屏幕
5. **快速部署** - 静态网站，易于部署和维护

## 📁 项目结构

```
it-tech-service/
├── index.html          # 主页面
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── main.js        # JavaScript功能
├── images/            # 图片资源目录
└── README.md          # 项目说明
```

## 🛠️ 技术栈

- **HTML5** - 语义化标记
- **CSS3** - 现代样式和布局
- **JavaScript (ES6+)** - 交互功能
- **Font Awesome** - 图标库
- **Google Fonts** - 字体服务

## 🚀 快速开始

### 本地运行
1. 克隆或下载项目
2. 直接打开 `index.html` 在浏览器中查看

### 在线部署
1. 上传到任意静态网站托管服务：
   - GitHub Pages
   - Netlify
   - Vercel
   - 阿里云OSS
   - 腾讯云COS

2. 访问部署后的URL即可

## 📝 使用说明

### 修改内容
1. **联系方式**: 修改 `index.html` 中的电话和邮箱
2. **服务价格**: 更新 `index.html` 中的价格信息
3. **案例内容**: 替换成功案例为真实项目
4. **样式定制**: 修改 `css/style.css` 中的颜色变量

### 添加图片
1. 将图片放入 `images/` 目录
2. 在HTML中更新图片路径
3. 建议使用WebP格式优化加载

## 🚨 注意事项

1. **表单数据**: 当前使用localStorage存储，生产环境需要后端API
2. **联系方式**: 确保电话和邮箱正确
3. **响应式测试**: 在不同设备上测试显示效果
4. **性能监控**: 定期检查页面加载速度
