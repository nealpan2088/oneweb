❌ 错误1: SyntaxError: Unexpected token 'export'

文件: wifi-modal.js:383:1

原因: ES6模块语法在普通script标签中不支持

状态: 已修复

✅ 修复方案:

  • 移除 export default WifiModal 语句
  • 使用传统的全局变量方式暴露类
  • 确保HTML页面可以正常访问 window.WifiModal
// 修复前（错误）: export default WifiModal; // 修复后（正确）: if (typeof window !== 'undefined') { window.WifiModal = WifiModal; window.showWifiModal = function(wifiInfo, onClose) { return WifiModal.showWifiInfo(wifiInfo, onClose); }; }

❌ 错误2: HTTP 429 Too Many Requests

端点: GET /api/shop/3/stats

原因: API请求过于频繁，触发后端限流保护

状态: 已修复

✅ 修复方案:

  • 修改数据加载逻辑：串行加载关键数据，并行加载次要数据
  • 使用 Promise.allSettled() 处理可能失败的请求
  • 添加错误处理和降级方案
  • 非关键API调用使用非阻塞方式
// 修复前（容易触发限流）: const [shopInfo, reviews, stats] = await Promise.all([ shopService.getShopInfo(), shopService.getReviews(), shopService.getStats() // 这个端点有严格限流 ]); // 修复后（更健壮）: // 1. 先加载关键数据 const shopInfo = await shopService.getShopInfo(); updateShopInfo(shopInfo); // 2. 并行加载次要数据（使用allSettled） const [reviews, stats] = await Promise.allSettled([ shopService.getReviews(), shopService.getStats() ]); // 3. 分别处理成功和失败 if (reviews.status === 'fulfilled') updateReviews(reviews.value); if (stats.status === 'fulfilled') updateStats(stats.value);
