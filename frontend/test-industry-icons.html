📱 响应式设计

行业图标适配各种设备屏幕：

  • 桌面端: 图标+文字水平排列，图标尺寸较大
  • 平板端: 适当缩小图标，保持水平排列
  • 手机端: 图标+文字垂直排列，优化触摸体验
/* 响应式设计 */ @media (max-width: 768px) { .shop-title-with-icon { flex-direction: column; /* 垂直排列 */ gap: 8px; } .shop-name-text { font-size: 1.5em !important; /* 缩小文字 */ } .industry-icon { font-size: 1.5em !important; /* 缩小图标 */ width: 40px !important; height: 40px !important; } }
