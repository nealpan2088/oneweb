## 🔍 快速查找技巧

### **使用grep查找**
```bash
# 查找特定文本
grep -n "将创意转化为代码" index.html

# 查找所有标题
grep -n "class=\"demo-title\"" index.html

# 查找所有链接
grep -o 'href="[^"]*"' index.html

# 查找所有图标
grep -o 'class="[^"]*fa[^"]*"' index.html
```

### **使用sed批量替换**
```bash
# 替换所有"GitHub"文本
sed -i 's/GitHub/GitHub仓库/g' index.html

# 替换所有邮箱
sed -i 's/panhaiping@example.com/your-email@domain.com/g' index.html

# 替换所有占位链接
sed -i 's|https://github.com/yourusername|https://github.com/real-username|g' index.html
```
