修复方案: 修改CSS媒体查询，强制手机端保持水平排列。

🔧 修复的CSS代码

/* 原代码（有问题） */ @media (max-width: 768px) { .shop-title-with-icon { flex-direction: column; /* ❌ 垂直排列 */ gap: 8px; } } /* 修复后的代码 */ @media (max-width: 768px) { .shop-title-with-icon { flex-direction: row !important; /* ✅ 强制水平排列 */ gap: 10px !important; flex-wrap: wrap; /* 空间不够时换行 */ justify-content: center; align-items: center; } .industry-icon { flex-shrink: 0; /* ✅ 防止图标被压缩 */ } /* 超小屏幕优化 */ @media (max-width: 480px) { .shop-title-with-icon { gap: 8px !important; } } }

📐 响应式设计原则

  • 水平排列优先: 图标和文字保持在同一行
  • 适当间距: 根据屏幕尺寸调整间距
  • 防止压缩: 使用 flex-shrink: 0 保护图标
  • 弹性换行: 空间不足时自动换行
  • 居中对齐: 保持视觉平衡
